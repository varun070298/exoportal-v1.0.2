The implementation process is based on the ULM diagrams developed during the design phase. For each client, the following developments are to be made in order to build the personalised solution:

  • The server parametrisation in order to meet with the security policy of the client
  • Development of the access to the organisation model of the client company (LDAP, SGBD)
  • Connectors development using JCA
  • Implementation of the portlets to access third party applications using the eXo platform™ portlet development framework
  • XML representation of the modelled business processes
  • Build of unit tests based on use cases defined in the design phase, according to software industries best practises and XP programmation.

Finally the eXo platform SARL performs the load stress tests before putting the portal into production.
