Blogs are a new fancy way for people to communicate and share ideas over the internet. Over the past 2 years the number of blogs have dramatically increased and its spreading probably helped eXo platform to have the success it has today. We have computed a list of the blogs that dealt with the eXo platform, if you know others feel free to mail us at blogs@exoplatform.com

We would like to start with Punit Pandey's Weblog on Portlets. This blogs contains valuable information on portlets (portlet API, WSRP and portals in general). Punit has also created two Yahoo groups :

  • Portlet Discussion Group
  • WSRP Discussion Group

Carlos Perez's weblog - called Manageability - is also full of precious information including quite a big list of Open Source portals. A poll "what is the best open source portal server" is also available.

Our portal is based on Java Server Faces that is the web framework new standard. For information on web components and products available have a look at JSF Central. James Holmes' site is also interesting for JSF beginners. Finally, have a look at Kito Mann TSS interview on Java Server Faces.

Our software comes with an eclipse plugin. For more plugins have a look at the eclipse plugin repository.

Each time we announced news, articles or released we created a thread on The Server Side site. Many comments and questions where asked there, so it may interest you :
