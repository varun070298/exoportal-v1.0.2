Blog 是一个特别的方式使人们通过互联网来沟通和分享思想。在过去的两年里,blog的数量令人吃惊的增长，并且其传播在一定程度上帮助eXo 平台获得它今天的成功。我们统计了一张和eXo 平台相关的blog列表，如果您知道其他的，请不吝写信给以下地址告知我们。blogs@exoplatform.com

我们从Punit Pandey's Weblog on Portlets开始. 这篇 blog 包含了关于portlet的有价值的信息 (portlet API, WSRP 和 portals 概述). Punit 同时创建了两个 Yahoo groups :

  • Portlet Discussion Group
  • WSRP Discussion Group

Carlos Perez's weblog - 叫做 Manageability - 同样包括很多宝贵的信息，包括一个庞大的开元门户的列表。 一项调查 "什么是最好的开源门户" 同样可以获得。

我们的门户是基于 Java Server Faces ，它是 web framework 的新的标准。想得到关于web components 和可用产品的信息，请浏览JSF Central. James Holmes' site 同样对于JSF初学者很有趣。 最后, 看一下 Kito Mann 接受 TSS 对Java Server Faces的访问

我们的软件包括了一个eclipse 插件。 想得到更多的插件，请访问 eclipse plugin repository.
