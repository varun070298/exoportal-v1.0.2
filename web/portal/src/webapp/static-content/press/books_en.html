The portlet API and WSRP are quite new technologies and few books where published. As The eXo platform™ was one of the very first portal to be certified we obtained a great momumtum and the first book about JSR 168 portals based all its code examples on our software. We did not write any line of this book so all the great comments about our architecture were not influenced.

This book does not only contain usefull information about the portlet API but also many interesting technologies that are usefull while developing its custom enterprise portal.

Here is the agenda :

  • Lucene
  • James (Mail Server)
  • Java Mail API
  • Jakarta Slide (WebDAV Content Management System)
  • Object Relational Bridge (OJB), with ODMG and JDO
  • Oasis Web Services Security (WSS)
  • Tomcat Security Realms
  • JSR 168 - Portlet API
  • SSL Security
  • Quality Assurance using Ant, Junit
  • Rational Unified Process
  • Extreme Programming (in the same chapter as RUP no less)
  • CRC Card Design

  • Model 1, Model2, and Model2x Architectures (2X what's that? Um.. Model 2 with some extra XML I guess)
  • GoF Patterns (MVC, Memento, Adaptor, etc)

  • Javascript (An entire chapter)
